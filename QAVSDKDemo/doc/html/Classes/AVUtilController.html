– audioDataComes:type:

音频数据透传回调。

- (QAVResult)audioDataComes:(QAVAudioFrame *)audioFrame type:(QAVAudioDataSourceType)type

Parameters

audioFrame

音频数据，输出类型从此参数读取数据，输入数据类型将数据写入此参数。
type

音频数据类型。 @remark 回调函数设定为专门处理数据用。函数回调在非主线程，请确保线程安全。特别是不要在回调函数中直接调用SDK接口。 成功返回QAV_OK, 否则返回QAV_ERR_FAILED。

Discussion

通过回调函数，来通知外部读取或者写入具体类型的音频数据。
